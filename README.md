# ANN_ILE_KREDI-KARTI_DOLANDIRICILIGI_TESPITI
Şiddetli dengesiz kredi kartı işlem verileri üzerinde ANN kullanılarak dolandırıcılık tespiti yapılmıştır. SMOTE ile veri dengelenmiş, Dropout ve EarlyStopping ile aşırı öğrenme kontrol altına alınmıştır. Model, dolandırıcılık işlemlerinin %87’sini doğru yakalayabilmekte ve ROC-AUC skoru 0.969’dur.
